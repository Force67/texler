\section{Introduction}

This is a demonstration of a multi-file LaTeX project compiled with Texler's Docker container.

\subsection{Background}

Multi-file LaTeX projects are common for large documents such as:
\begin{itemize}
\item Thesis and dissertations
\item Research papers with multiple sections
\item Books and technical documentation
\item Reports with separate chapters
\end{itemize}

\subsection{Purpose of This Demo}

This demo project showcases how the existing Texler backend handles multiple LaTeX files that are combined using the \texttt{\textbackslash include} command.
